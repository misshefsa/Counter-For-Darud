# Counter Web

A simple, responsive counter web app demonstrating component state/props, event handling, and styling (navy-blue theme). Includes click and keyboard controls plus console logging for easy debugging — perfect for learning or showcasing front-end basics.

## Features
- Increase / decrease / reset counter
- Console logging of each counter update (for debugging / learning)
- Responsive layout and navy-blue color theme
- Small, well-documented code (works in Angular, React, or plain JS — pick your flavor)
